# Stack

> 참고 : https://gmlwjd9405.github.io/2018/08/03/data-structure-stack.html

## 개념

- 한쪽 끝에서만 자료를 넣고 뺄 수 있는 LIFO 형태의 자료구조
  - Top에서 삽입과 삭제가 이뤄진다.



## 특징

- LIFO : 후입선출
- 선형자료구조 : 자료 간의 관계가 1대1 관계를 가짐



## 활용

- 재귀 알고리즘
- DFS (깊이 우선 탐색)
- 웹 브라우저 방문기록 (뒤로 가기)
- 실행 취소 (Undo)
- 후위표기법
